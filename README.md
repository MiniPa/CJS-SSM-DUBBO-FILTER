# 简介
dubbo日志扩展插件，方便定位并排查异常。

# 安装

```
git clone https://github.com/crossoverJie/SSM-DUBBO-FILTER.git
```

```
cd SSM-DUBBO-FILTER
```

```
mvn clean
```

```
mvn install
```

# 使用

```xml
<dependency>
    <groupId>com.crossoverJie</groupId>
    <artifactId>SSM-TRACE-FILTER</artifactId>
    <version>1.0.0</version>
</dependency>
```
